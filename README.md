# Bucket Monte-Carlo

## Compile

```
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build . --target install
```

## Usage

```
$ mkdir work
$ cd work
$ ../bin/bucket
```

## Output

A csv file is created.

| header | unit | description |
| ------ | ---- | ----------- |
| nref   |      | # of reflection |
| time   | ns   | photon detection time |
